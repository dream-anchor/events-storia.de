# SEO Strategy - STORIA Events & Catering

## Overview

This document outlines the SEO and GEO (Generative Engine Optimization) strategy for events-storia.de.

---

## Current Implementation

### Technical SEO
- SSG (Static Site Generation) for all public pages
- JSON-LD structured data (FAQPage, LocalBusiness, etc.)
- Sitemap at `/sitemap.xml`
- Canonical URLs
- Mobile-responsive design
- Core Web Vitals optimized

### On-Page SEO
- Meta titles and descriptions per page
- H1/H2 heading hierarchy
- Alt text for images
- Internal linking structure

### Local SEO
- LocalBusiness schema
- NAP consistency (Name, Address, Phone)
- Service area: Munich and surrounding areas

## GEO Strategy (Generative Engine Optimization)

### Principles
1. Clear, factual answers to common questions
2. Entity-rich content (locations, services, prices)
3. Authoritative voice with specific details
4. FAQ content optimized for AI comprehension

### Implementation
- FAQ page with comprehensive Q&A
- Service descriptions with specific details
- Location mentions throughout content
- Price transparency
